# src/agent/llm/schema.py

Purpose
- Defines schemas for LLM inputs/outputs and provides helpers to validate or coerce responses into structured data.

Key features
- Prompt and response schema definitions (types of expected fields, validators).
- Converters to and from prompt-safe formats.

Where used
- Used by response_generator and extractor to enforce consistent formats.
